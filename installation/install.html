Introduction of UI-Book
UI-Book is an open-source style architecture book, which helps the programmer to use the best styles and components, using preset class names. UI-Book helps you build your dream project quickly, concentrating a lot on functionality because we care about your designs.
Installation Guide
In order to use UI Book in your project, Copy the Link given below and paste it in the <head> tag of your HTML, above all other style sheets. 
            <link rel="stylesheet" href="https://uibook.netlify.app/css/components.css">
