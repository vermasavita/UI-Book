List
Lists are continuous, vertical display of text or images.
Simple List
Simple list can be created by using class = "list-box".
    Book Genres
  • Fiction
  • Non-finction
  • Fantasy
  • Triller
  • Romance
                        <div class="list-box">

                            <ul>
                                <div class="list-title">Book Genres</div>
                                <li>Fiction</li>
                                <li>Non-finction</li>
                                <li>Fantasy</li>
                                <li>Triller</li>
                                <li>Romance</li>
                            </ul>

                        </div>
Nested List
Nested list means list inside a list. It can be created by using class = "nested-list" inside the existing class.
    Book Genres
  • Fiction
    • Harry Potter
    • Meluha
    • The Archer
  • Non-finction
  • Fantasy
  • Triller
  • Romance
                        <ul>
                            <div class="list-title">Book Genres</div>

                            <li class="nested-list">Fiction
                                <ul>
                                    <li><a href="#">Harry Potter</a>
                                    </li>
                                    <li><a href="#">Meluha</a></li>
                                    <li><a href="#">The Archer</a></li>
                                </ul>
                            </li>

                            <li>Non-finction</li>
                            <li>Fantasy</li>
                            <li>Triller</li>
                            <li>Romance</li>

                        </ul>
Stacked List
Stacked list can be created by using class = "stacked-list".
  • Fiction Book
    "Fiction" refers to literature created from the imagination.
  • Non Fiction
    Nonfiction is any document or media content that intends, in good faith, to present only truth and accuracy regarding information, events, or people. 
                        <ul class="stacked-list">

                            <li>
                                <div class="card-title">Fiction Book</div>
                                <div class="card-content">"Fiction" refers to literature created from the imagination.
                                </div>
                            </li>

                            <li>
                                <div class="card-title">Non Fiction</div>
                                <div class="card-content">Nonfiction is any document or media content that intends, in good
                                    faith, to present only truth and accuracy regarding information, events, or people.
                                </div>
                            </li>

                        </ul>
