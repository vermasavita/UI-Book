Image
Images can be displayed for different purposes on websites and applications in different sizes.
Full screen image
It can be created by using class = "img-box-1" and use img tage inside that. 
                        <div class="img-box-1">
                            <img src="/images/sky-img.jpeg" alt="sky">
                        </div>
Round Image
This round and responsive image can be created by using class = "img-box-2" and use img tag inside that. 
                        <div class="img-box-2">
                            <img src="/images/sun-img.jpeg" alt="sun">
                            <img src="/images/moon-img.jpeg" alt="moon">
                        </div>
