Avatar
In simple words, Avatar is the profile picture we put on our profile information page, navigation bar, and so on.
Image Avatar
Image avatar can be created by using img, img, src, srcSet to the component. 
            <div class="avatar-box">

              <img src="/images/img-avatar-1.jpeg" alt="avatar image 1" class="avatar">
              <img src="/images/img-avatar-2.jpeg" alt="avatar image 2" class="avatar">
              <img src="/images/img-avatar-3.jpeg" alt="avatar image 3" class="avatar">

            </div>
