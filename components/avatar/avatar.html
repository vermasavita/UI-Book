Avatar
In simple words, Avatar is the profile picture we put on our profile information page, navigation bar, and so on.
Profile Avatar
Profile picture can be created using class = “avatar”, inside the img tag. 
            <div class="avatar-box">

              <img src="/images/img-avatar-1.jpeg" alt="avatar image 1" class="avatar">
              <img src="/images/img-avatar-2.jpeg" alt="avatar image 2" class="avatar">
              <img src="/images/img-avatar-3.jpeg" alt="avatar image 3" class="avatar">

            </div>
Square Avatar
Size of avatar can be changed adding class = “square-avatar” in existing class called "avatar", inside the img tag. 
            <div class="avatar-box">

              <img src="/images/img-avatar-1.jpeg" alt="avatar image 1" class="avatar square-avatar">
              <img src="/images/img-avatar-2.jpeg" alt="avatar image 2" class="avatar square-avatar">
              <img src="/images/img-avatar-3.jpeg" alt="avatar image 3" class="avatar square-avatar">

            </div>
Size Avatar
Size of Avatar can be changed by using class = "size-1" / "size-2" / "size-3" to the avatar class.
size-1 stands for small | size-2 stands for medium | size-3 stands for large 
            <div class="avatar-box">

              <img src="/images/img-avatar-1.jpeg" alt="avatar image 1" class="avatar size-1">
              <img src="/images/img-avatar-2.jpeg" alt="avatar image 2" class="avatar size-2">
              <img src="/images/img-avatar-3.jpeg" alt="avatar image 3" class="avatar size-3">

            </div>
