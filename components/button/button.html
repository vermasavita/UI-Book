Button
Button is the container which used to perform specific action.
Basic Button
Basic buttons have three different styles. Use text style by using class = "text", container button by using class = "contained" and outline button by using class = "outline. 
            <button class="btn text">Text</button>
            <button class="btn contained">Contained</button>
            <button class="btn outline">Outline</button>
Link Button
Link buttons have links inside them. It is created by using an anchor tag instead of a button tag. Use the link button by using class = "link". 
            <button class="btn">
              <a class="link" href="#">Link</a>
            </button>
Icon Button
Icon button gets created by using an icon inside the button. It can be created this by using class = "btn-icon" along with class = "btn". 
            <button class="btn btn-icon">
              <i class='bx bxs-send'></i>
            </button>
Floating Action Button
Floating button used to float the screen generally. This can be created by using class = "float-icon". 
            <button class="float-icon">
              <i class='bx bx-plus'></i>
            </button>

            <button class="float-icon">
              <i class='bx bx-up-arrow-alt'></i>
            </button>
