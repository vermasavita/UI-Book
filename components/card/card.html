Card
Card provides a conatiner to keep the text, image, button and icon etc all together.
Text Card
Text card only contain text. It can be created by using three classes i.e class = "card-box", class="card-title", class="card-content".
Kung Fu Panda
You are the Master of your own Destiny: No one and nothing can come in between you and you destiny except you. 
                        <div class="card-box">

                            <div class="card-title">Kung Fu Panda</div>

                            <div class="card-content">You are the <strong>Master</strong> of your own
                                <strong>Destiny</strong>:
                                No one and nothing can come in between you and you destiny except you.
                            </div>

                        </div>
Text Overlay | Image | Dismiss Card
To add an image on a card use img tag, for text overlay use class = "txt-overlay" and to cancel the card use icon inside class = "card-box".

Badge Card
